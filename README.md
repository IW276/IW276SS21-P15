* Python 3.6 (or above)
* OpenCV 4.1 (or above)
* Jetson Nano
* Jetpack 4.5
* Tensorflow 2.3 (or above)
> [Optional] ...

## Running

Open the terminal and navigate to the project folder. Then execute the following commands:

```
sh build-docker-image.sh

sudo docker run -it as15_my_img

sh run-tracker.sh
```
